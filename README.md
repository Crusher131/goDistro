## Go-Distro

GolangでOSとそのバージョン、コードネームを取得するためのライブラリです。

現状では自分が使っているOSしかサポートしていません。

### License

### Sample tool

You can chck your local environment with following command.

```bash
go run ./cmd/main.go
```
