## Go-Distro

Golang で OS とそのバージョン、コードネームを取得するためのライブラリです。

現状では自分が使っている OS しかサポートしていません。

結構汎用性の高い構造似できたと思うので、ディストリビューション固有のコードを追加してくれると助かります。
